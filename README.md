### Основа map-reduce

Программа работает в асинхронном режиме. В ней существует два типа задач. 
 * ***map*** - трансформация входящих данных. В данном случае это текст, который разбивается на слова и представляется в виде пары key - value, где key это слово, а value - это цифра 1.
Слова обрабатываются в отдельном потоке, и записываются в отдельные промежуточные файлы.
 * ***reduce*** - это суммация, подсчет данных слов из всех промежуточных файлов в сумму всех слов и запись результат в файл корня проекта *output/result-sum.txt*.

## Запуск
1. ```git clone https://github.com/SidorinAA/map-reduce.git```
2. Запустить ```gradle build```
3. Запустить программу ```java -jar build/libs/map-reduce-1.0-SNAPSHOT.jar path/*.txt  path/*.txt n m```
где после указании файла jar указывается массив файлов для обратоки. Параметер n - указывает кол-во потоков для map задачи, а m - кол-во потоков для reduce задач
